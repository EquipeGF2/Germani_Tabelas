# NCM categorias

## Objetivo
Manter o vínculo entre tipos de produto e códigos NCM para Dallas.

## Quando usar
- Criar novas categorias fiscais.
- Subir a base padrão de Dallas via seed.

## Passo a passo
1. Selecione a empresa.
2. Cadastre **Nome** e **NCM** e clique em **Adicionar**.
3. Para carga inicial, clique em **Seed Dallas** (ignora duplicados).

## Erros comuns
- Deixar NCM vazio ou com formatação incorreta.

## Glossário
- **NCM**: Nomenclatura Comum do Mercosul utilizada para tributação.
